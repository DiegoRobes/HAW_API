Hugo Award Winners

Get nominees and winners from every year

Documentation

In this API service, you can get the Hugo Award winners and nominees from all categories, and every year since each has been established.
We are currently working on the 'Retro Hugos', so this document will be updated in time.

One-for-all route

Every request on this site is worked using the route "/request/{arguments}", the arguments you'll need to use will be clear below.

Arguments
"/request/{year}..." From 1953 to 2022. Take on account that only Novel category started on 1953, the other categories started later.
"/request/1985?category={category}..." We offer the 4 main categories of the Hugo Awards: novel, novella, novelette, short_story. All lowercase. Mind the "_" on short_story!
"/request/1985?category=novel&api-key={your-secret-key}" You can get it down below, once you sign up by sharing an email.
Note: We will never share your email with anyone.

Results

When a request is done the right way, and the server has gathered the information, the results are delivered in a Json format. Note that the winner is always the first result and is marked with an asterix.
